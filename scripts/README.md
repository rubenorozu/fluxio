# Scripts

Este directorio contiene scripts de utilidad para el proyecto.

## Scripts disponibles:

- `seed-database.ts` - Seed inicial de la base de datos
- `seed-production.ts` - Seed para producción
- `migrate-to-multitenant.ts` - Migración a multi-tenant
- `normalize-emails.ts` - Normalizar emails en la base de datos
- `create-superuser-ruben.sql` - SQL para crear superusuario

## Nota

Los scripts de TypeScript deben ejecutarse con `npx ts-node` o agregarse como scripts en `package.json`.
